# Projet IA - ESGI

## Description
Ce projet a été développé dans le cadre du cours d'intégration de l'IA à l'ESGI. Il utilise l'API Inference de Hugging Face pour effectuer des appels au modèle DeepSeek.

## Installation

1. Cloner le repository
2. Installer les dépendances :
```bash
npm install
```

3. Ajouter votre clé API Hugging Face dans le fichier `.env`

## Lancement en développement

```bash
npm run dev
```

Le serveur sera accessible sur `http://localhost:3000`

## Technologies utilisées
- Node.js
- API Inference Hugging Face
- Modèle DeepSeek-V3-0324
